Flood Probability
Prediction System

A machine learning pipeline that ingests 11 socio-environmental risk indicators and outputs a continuous flood probability score between 0 and 1.

Project Brief

What does this system do?

Flooding is a leading natural disaster, driven by a complex interplay of meteorological, geographical, and anthropogenic factors. This dashboard visualises a Linear Regression model trained on the Flood Prediction Dataset to predict flood probability from 11 engineered risk features.

The goal is to give researchers and planners an interpretable, interactive tool — not a black box. Every design decision (feature selection, preprocessing, split strategy) is explained in the sections below.

Methodology Note

Why correlation-based feature selection?

Pearson correlation is a fast, interpretable filter method. For a linear model specifically, features with near-zero linear correlation with the target add noise without improving predictions. We removed features with |r| < 0.05 to keep the model parsimonious.

All 11 features — MonsoonIntensity, TopographyDrainage, RiverManagement, Deforestation, Urbanization, ClimateChange, DamsQuality, Siltation, AgriculturalPractices, Encroachments, and IneffectiveDisasterPreparedness — passed the threshold and were retained.

Train–Test Split

80 / 20 Stratified Split — Why?

Train 80%
Test 20%

An 80/20 ratio gives the model sufficient training data while reserving enough unseen samples for a reliable generalisation estimate. random_state=42 ensures reproducibility across runs — any collaborator running the same notebook will get identical splits.

Step 04

Model Analytics

How well does the Linear Regression model generalise? Evaluation metrics and diagnostic charts are shown below.

Interpreting the Metrics

What do MAE, MSE, RMSE, and R² tell us?

MAE = 0.031

On average, predictions deviate by ±3.1 percentage points from the true probability — a tight error for a 0–1 scale.

RMSE = 0.038

RMSE penalises large errors more heavily than MAE. A value close to MAE indicates few large outlier errors.

MSE = 0.0015

The squared average error. Used as the actual loss function optimised during Linear Regression training.

R² = 0.335

The model explains ~33.5% of variance in flood probability. This is modest — non-linear models or richer features could improve this further.

Actual vs. Predicted (Scatter)

Points near the diagonal line indicate accurate predictions. Spread indicates residual error.

Residual Distribution

A bell-shaped residual distribution centred at zero confirms that the model's errors are random, not systematic.

Feature Correlation Matrix

Pairwise Pearson correlations between all features and the target. Strong off-diagonal values may indicate multicollinearity.
